Мощный и гибкий плагин, который заменяет стандартную систему уровней и опыта
в Minecraft на кастомизируемую и масштабируемую. 
Подходит для RPG-серверов, экономических проектов и любых других типов серверов, 
где важна индивидуальная прогрессия игрока.
{{< tabs "tabsId" >}}

{{< tab "Гибкая система уровней" >}}
Настройте количество уровней, формулу получения опыта, множители за действия (добыча блоков, убийства мобов, крафт и т.д.).
{{< /tab >}}

{{< tab "Кастомные источники XP" >}}
- выполнение квестов
- голосование
- онлайн-время
- взаимодействие с NPC / командами
- кастомные события через API
{{< /tab >}}

{{< tab "Статистика" >}}
встроенные команды /level, /xp, /toplevels + API для интеграции с другими плагинами (например, Scoreboard, PlaceholderAPI, MMOItems).
{{< /tab >}}

{{< tab "Лучший выбор?" >}}
> код полностью оптимизирован, имеется включаемый debugger с логами в консоль событий получения XP, повышения уровня и ошибок конфигурации, оформление уровней в виде мапы - быстрота, а не ваши листы.
{{< /tab >}}

{{< /tabs >}}

## Совместимости:

- Paper 1.20.x
- MythicMobs для опыта с кастомных мобов и нового дропа
- MMOItems для новых предметов, связанных с уровнем игрока
- PlaceholderAPI для плейсхолдеров
- Quests для настройки получаемого опыта квестами
